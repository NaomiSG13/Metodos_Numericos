# Tema 2: Métodos Numéricos para Solución de Ecuaciones

Este módulo se enfoca en encontrar las raíces de funciones mediante diversos algoritmos:

* ✅ **Bisección**
* ✅ **Falsa Posición**
* ✅ **Newton**
* ✅ **Secante Cerrado**
* ✅ **Secante Abierto**

Cada método incluye su análisis de convergencia y el número de iteraciones necesarias para alcanzar la tolerancia deseada.
