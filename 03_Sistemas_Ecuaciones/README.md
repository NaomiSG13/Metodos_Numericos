# Tema 3: Sistemas de Ecuaciones Lineales

Implementación de algoritmos para resolver sistemas de la forma $Ax = B$:

### Métodos Directos:
* 🔹 **Eliminación Gaussiana**
* 🔹 **Método de Gauss-Jordan**

### Métodos Iterativos:
* 🔸 **Método de Gauss-Seidel**
* 🔸 **Método de Jacobi**

*Se incluye un análisis de resultados comparando la eficiencia y estabilidad de cada método.*
